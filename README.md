# project-templates

Onsen UI project templates for Cordova and Visual Studio 2015.

## Build

```
$ npm install gulp -g
$ npm install
$ gulp build [--vsix <version>]

// Only Cordova templates
$ gulp build-cordova
// Only VS2015 templates
$ gulp build-VS2015 [--vsix <version>]
// Only MobileFirst Platform templates
$ gulp build-MFP
// Only TACO templates
$ gulp build-TACO


```
